Aura Art Collective

Project Overview
Aura Art Collective is a full-stack e-commerce platform designed for users to browse, purchase, and manage digital images or artwork. It features a robust architecture built on the MERN-Stack principles (with MySQL instead of MongoDB), providing a seamless shopping experience for customers and powerful management tools for administrators.

Key Goals

1.  Provide a responsive and intuitive shopping interface.

2.  Securely handle user authentication, shopping cart state, and order processing.

3.  Offer a dedicated administrative portal for comprehensive order and site management.

Features
User / Customer Experience (Frontend)

1.  Product Browsing: Navigate the shop to view available images.

2.  Shopping Cart: Add, update, and remove items from the cart, with state persistence (local or database-backed).

3.  Authentication: Secure user registration, login, and logout.

4.  Order History: View personal history of past purchases.

5.  Checkout Flow: Submit new orders via the API.

Administration Portal (Backend & Admin Frontend Views)

1.  Comprehensive Order Management: Retrieve all system orders for monitoring and fulfillment.

2.  Order Status Updates: Update the status of customer orders (e.g., Pending, Shipped) via API endpoints.

3.  KPI Metrics: Access key performance indicators (KPIs) and site usage metrics.

4.  Contact Message Handling: View and delete contact messages submitted by users.

Technology Stack

1.  Frontend -> React (with Vite) Fast, component-based user interface.
2.  Styling -> Tailwind CSS , Utility-first CSS framework for rapid, responsive styling.
3.  Backend/API -> Node.js & Express.js , High-performance, unopinionated server framework.
4.  Database -> MySQL , Relational database used for structured storage.
5.  ORM -> Sequelize , Promise-based ORM for easy interaction with MySQL (managing Product, User, CartItem, Order models).
6.  Authentication -> JWT (JSON Web Tokens) , Secure token-based authorization for all API requests.
